Book Post Module intallation instructions

1.  Copy the "bookpost" directory into a Drupal modules directory,
    like sites/all/modules.

2.  Enable module at: Administer->Site Building->Modules.

3.  Turn on input filter at: Administer->Site Configuration->Input
    Formats.  Open any input filter (for example, Filtered HTML) and
    check box for Book Post.

This module requires the JSON and cURL PHP libraries.  These are
compiled by default with PHP 5.2.
